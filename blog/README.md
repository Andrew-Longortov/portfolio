# /blog

Задел на будущее. Когда дойдёшь до бэкенда (Django/FastAPI или Node.js + БД),
это хорошее место, чтобы превратить статичный блог в полноценный CRUD:
статьи из базы данных, создание/редактирование через API.

Пока можно оставить пустым или сделать статичную версию на чистом HTML —
по желанию, не обязательный этап для первой версии портфолио.
